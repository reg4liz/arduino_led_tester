# arduino_led_tester
LED tester on atmega328 with adjustable current source.

## Glorious prototype
<img src="https://github.com/reg4liz/arduino_led_tester/blob/68fc972c25ec32f19a67384242dea3d9fa7e4213/img/1.jpg" alt="Prototype" width="300"/>

## Main components used for prototype
* [ATMEGA328P](https://www.digikey.com/en/products/detail/microchip-technology/ATMEGA328P-PU/1914589)
* [MCP6001 op-amp](https://www.digikey.com/es/products/detail/microchip-technology/MCP6001T-I-OT/551760)
* [SI2302CDS nmos](https://www.digikey.com/en/products/detail/vishay-siliconix/SI2302CDS-T1-GE3/1978729)
* [PEC12R-4025F-N0024 encoder](https://www.digikey.es/es/products/detail/bourns-inc/PEC12R-4025F-N0024/4499645)
* [ADS1115 ADC Module](https://aliexpress.com/item/1005006532275889.html)
* [MCP4725 DAC Module](https://aliexpress.com/item/1005007384695140.html)
* [SSD1306 128x64 OLED display](https://aliexpress.com/item/1005004355547926.html)
