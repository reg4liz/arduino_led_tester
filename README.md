# arduino_led_tester
LED tester on atmega328 with adjustable current source.

## Glorious prototype
<img src="https://github.com/reg4liz/arduino_led_tester/blob/68fc972c25ec32f19a67384242dea3d9fa7e4213/img/1.jpg" alt="Prototype" width="300"/>

## Components
* [ADS1115 ADC Module](https://aliexpress.com/item/1005006532275889.html)
* [MCP4725 DAC Module](https://aliexpress.com/item/1005007384695140.html)
* [MCP6001 op-amp](https://www.digikey.com/es/products/detail/microchip-technology/MCP6001T-I-OT/551760)
